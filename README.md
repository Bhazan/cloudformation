Collection of cloudformation template 

## Lets journey begins

### Template anatomy

#### JSON format
```
"AWSTemplateFormateVersion" : "version date",

"Description" : "JSON String",

"Metadata" : {
    template metadata
},

"Parameters" : {
    set of parameters
},

"Rules" : {
    set of rules
},

"Mappings" : {
    set of mappings
},

"Conditions" : {
    set of conditions
},

"Transform" : {
    set of transform
},

"Resources" : {
    set of resources
},

"Outputs" : {
    set of outputs
}
```
